FORMAT RULES:

1. **Action Format:**
   - Click "Element Name"
   - Type "value" into "Field Name"
   - Select "option" from "Dropdown Name"
   - Verify "Expected Text"
   - Clear "Field Name"

2. **Wait Times:**
   - After navigation: Wait for 1-2 seconds
   - After form submission: Wait for 3 seconds
   - After heavy operations: Wait for 30 seconds
   - After field entry: Wait for 1 seconds

3. **CSV Data Placeholders:**
   - Use format: %ColumnName%
   - Add comment: -- Replace with values from csv
   - Examples: %Username%, %Password%, %Counties Served%

4. **Comments:**
   - Use -- for inline comments
   - Examples: -- Generate CSV, -- Test with invalid data

5. **Verification:**
   - Verify "Exact text to find"
   - Verify "Field Name" field is visible
   - Verify "Field Name" field has error styling
   - Verify no error message is displayed

6. **Field References:**
   - Always use exact field labels in quotes
   - Examples: "Counties Served", "Full Patient Panel", "Changes to Target Population Measures"
