FORMAT RULES:

1. **Section Headers:**
   - Use # for main sections
   - Examples: # Login, # Navigate to Form, # Complete Form Pages

2. **Navigation:**
   - Navigate to [URL]
   - Click "[Element Name]"
   - Wait for X seconds

3. **Data Entry:**
   - Type "value" into "Field Name"
   - Type "%CSV Column%" into "Field Name" (for CSV data)
   - Select "option" from "Dropdown Name"
   - Check "Checkbox Name" checkbox

4. **Verification:**
   - Verify "Expected text or element"
   - Verify "Field Name" field has error styling
   - Verify no error message is displayed

5. **Wait Times:**
   - After navigation: Wait for 1-2 seconds
   - After form submission: Wait for 3 seconds
   - After heavy operations (CSV generation): Wait for 30 seconds
   - After field entry: Wait for 1 seconds

6. **Comments:**
   - Use # for comments
   - Example: # This is a comment

7. **Field Names:**
   - Always use exact field labels in quotes
   - Include full field descriptions
   - Examples: "Number of people on listserv", "Counties Served"

8. **CSV Placeholders:**
   - Format: %ColumnName%
   - Examples: %Username%, %Password%, %Report ID%, %Counties Served%
